Parva is my (blue_flycatcher) computer <3

## v0.1

* 24-bit architecture
* 8 registers
* 120Hz
* 128 words of memory (384 bytes), expandable, 24-bit addressing
* Doubleword memory read/write operations
* Bitshift, multiply, divide instructions
* 4 I/O ports
* Note-based pixel display
